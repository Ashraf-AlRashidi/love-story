# Love Story — تحسينات مبدئية

أضفت مجموعة صغيرة من التحسينات كمثال عام:

- صفحة رئيسية بسيطة `index.html` تحتوي على إعدادات العرض للهاتف وتحسينات أساسية.
- ملف أنماط `css/styles.css` بسيط لتنسيق الصفحة.
- ملف README مختصر يصف التغييرات.

إذا تحب أي تغييرات إضافية (نصوص، صور، ترجمة/تنسيق) أخبرني وسأعدل في نفس الفرع أو أنشئ فرع آخر.
