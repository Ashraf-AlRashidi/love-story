# ❤️ Love Story Generator

موقع رومانسي لإنشاء قصة حبك الخاصة وتشاركها مع حبيبك!

## 🎯 الميزات

✨ **انشاء موقع مخصص** - كل زوج يحصل على موقع بالبيانات اللي يختارها
💕 **تصميم رومانسي** - تأثيرات جميلة ومتحركة
🔐 **حماية برسمة دخول** - كلمة سر تحمي القصة
📸 **معرض صور** - عرض الذكريات بطريقة حلوة
🎵 **موسيقى خلفية** - أضيف الموسيقى اللي بتحبها
📅 **رسائل يومية** - 365 رسالة طول السنة
⏳ **عداد الأيام** - يحسب كم يوم من بداية الحب
🎤 **وضع السينما** - اسمع صوت حبيبك

## 🚀 كيفية الاستخدام

### 1. اذهب للموقع
```
https://ashraf-alrashidi.github.io/love-story/
```

### 2. املأ بيانات قصتك
- اسمك واسم حبيبك
- تاريخ بداية الحب
- رسالة الحب
- الأسباب (ليه هو/هي بالذات؟)
- صور وموسيقى

### 3. احصل على رابط
الموقع هيعطيك رابط فريد تشاركه مع حبيبك

### 4. شارك الحب ❤️
النسخ والرابط وبعتها!

## 📝 البيانات المطلوبة

**إلزامي:**
- اسمك
- اسم حبيبك
- تاريخ البداية
- الرسالة الرئيسية
- 3 أسباب لليه بتحبهم

**اختياري:**
- كلمة دخول
- موسيقى (رابط MP3)
- صور (روابط الصور)
- صوتك (رابط MP3)
- رسائل يومية

## 🔒 الأمان

البيانات تُشفر في الرابط نفسه باستخدام Base64، ما في قاعدة بيانات تخزنها.

## 💻 التطويير

### المتطلبات
- GitHub Pages (مفعل مسبقاً)
- أي متصفح حديث

### الملفات
- `index.html` - صفحة النموذج
- `love-page.html` - صفحة العرض

## 📱 ملاحظات

- الموقع يعمل على الهاتف والحاسوب
- روابط الصور والموسيقى لازم تكون HTTPS أو CORS مفعلة
- إذا الصور ما حملت، راح يظهر رسالة خطأ

## 🎨 التخصيص

تقدر تعديل الألوان والخطوط بسهولة من CSS:
```css
--primary: #ff0055;  /* لون الحب الرئيسي */
--dark: #120509;     /* لون الخلفية */
```

## 📞 الدعم

إذا في مشاكل:
1. تأكد من روابط الصور والموسيقى صحيحة
2. استخدم روابط HTTPS فقط
3. الرابط لازم يحتوي على `?data=` صحيح
